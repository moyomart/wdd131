Site Name

Software Devs Cookbook

  • This name represents a web-based platform that is focused on bringing together all the 'ingredients' one needs to learn and master the skillsets required to become a professional software developer.

Site Purpose

  • The site aims to be the one stop shop where individuals from novice to expert can gain valuable insight into the career as a software developer. It provides tips, useful resources and guidance on how to become an expert in the trade including assistance in preparing a portfolio of work that can be used to apply for jobs

Scenarios

  • What is a Software Developer?
  • Who can become a software Developer?
  • What Industry can l work in ?
  • What are the benefits of learning software development?
  • Where do l start if l want to learn?
  • What skills do l need to work on?
  • Where can l get a job after l learn these skills?
